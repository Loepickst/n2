基础礼貌 ➜ 特定对象的尊敬

Step 0 郑重语 & 美化语 (基础礼貌)

【郑重语】定义：

不考虑对方地位高低，为了保持“得体、礼貌”而使用的语言。即：让你的日语听起来像个文明的成年人。

1. 特殊郑重语

在服务业、商务接待、正式演讲中，通常会将「です/ます」替换成一些特殊的表达。

原词 升级版 💬 经典语境例句
ある (有/在) ございます 店员 トイレはあちらにございます。
(洗手间在那边。)
です (是) でございます 电话 はい、担当の田中でございます。
(您好，我是负责此项工作的田中。)
いい (好) よろしい 确认 ご注文は以上でよろしいでしょうか。
(您点的餐就是这些，没问题吧？)
どう (怎么样) いかが 推荐 食後のコーヒーはいかがですか。
(饭后来杯咖啡怎么样？)
今日 (今天) 本日 honjitsu 广播 本日はご来店ありがとうございます。
(感谢您今天的莅临。)

2. 美化语

为了让语言听起来“优雅、有教养”而添加前缀，对此进行美化。
（区别：尊他语是为了尊敬对方，而美化语是为了修饰自己。）

饮食类
  • お酒 (酒)
  • お米 (米)
  • お水 (水)
  • ご飯 (饭)
生活类
  • お風呂 (洗澡)
  • お部屋 (房间)
  • お化粧 (化妆)
  • お店 (商店)
一般加“お”的固定搭配

有些词如果不加“お”，意思会变或者显得不够礼貌，通常已经变成了单词的一部分。

お腹 (肚子) おかず (菜肴) おやつ (零食) おにぎり (饭团)

Step 1 尊他语 (抬高对方)

【尊他语】定义：

动作主语是“听话人”或“长辈/上级”。通过抬高这个动作，来表示对对方的尊重。

常见尊他句式

① お + 第一连用形 + になる 最标准/最正式

[回家] 帰る → お帰りになる

先生はもうお帰りになりました。

(老师已经回去了。)

② 【被动态】 (れる / られる) 最常用/较轻松

变形规则和“被动态”完全一样。尊敬程度相对适中，日常口语出现频率极高。

[读] 読む → 読まれる

社長は毎朝、新聞を読まれます。

(社长每天早上都会读报纸。)

③ お + 第一连用形 + です 状态描述

表示正在进行或持续拥有的状态。

[持有] 持つ → お持ちです

チケットはお持ちですか。

(您带着票吗？)

④ (汉语词) + なさる する的升级版

[担心] 心配する → 心配なさる

ご心配なさらないでください。

(请您不要担心。)

Step 2 自谦语 (压低自己)

【自谦语】定义：

动作主语是“我自己”或“己方人员”。通过压低自己的动作，来衬托对方的高大。

注意：一般要分清是“为了对方做(一类)”还是“自己礼貌地说(二类)”。

1. 涉及对方的自谦 一般有对象

逻辑： 我的动作是“做给对方看的”或者“影响到对方的”。如果对方不在场，这个动作就不成立。

お / ご + 第一连用形 + する / いたす

[持有] 持つ → お持ちする

重そうですね、お持ちします。

(看起来很重呢，我来帮您拿。 → 动作指向：您)

2. 体现自我谦逊 无需对象

逻辑： 动作不一定影响对方，只是为了“向听话人展示自己的谦逊”。哪怕是自我介绍、或者说自己的状态，也可以用。

特殊动词 / (汉语词) + いたす

[名字] 言う → 申す

田中と申します。

(我叫田中。 → 不涉及对方，但向听众表谦虚)

Step 3 特定句式表达

除上述的基础动词变形外，日语中存在一系列固定的句式结构。这些句式通过组合“使役”、“授受”或“特定补助动词”，来实现更高级别的自谦或尊他功能。

1. 描述自身行为 (郑重陈述) 自谦

当说话人向听话人陈述自己的情感、或某种自我行为时，使用「申し上げます」替代普通的「します/言います」，以表达郑重的语气。

お/ご + 名词 + 申し上げます

例：心よりお詫び申し上げます。

(由衷地表示歉意。)

例：厚くお礼申し上げます。

(致以深切的谢意。)

2. 请求允许自身行为 (【使役态】自谦) 自谦

通过使用“【使役态】 (让...)”与“授受动词 (我领受...)”的组合，表达“请允许我做某事”的含义。这是商务日语中陈述自己动作的一般标准形式。

【使役态】(さ)せて + いただけませんか

(询问对方是否允许)

本日、早退させていただけませんか。

(今天能允许我早退吗？)

【使役态】(さ)せて + いただきます

(宣告自己将做某事)

説明させていただきます。

(那么，请允许我说明一下。)

3. 描述他人行为 (书面语中顿) 尊他

お/ご + (第一连用形) + いただき、...

(承蒙您做某事，我方获益)

ご検討いただき、ありがとうございます。

(承蒙您探讨，非常感谢。)

お/ご + (第一连用形) + くださり、...

(您特意做某事，我方感激)

遠方よりお越しくださり、感謝いたします。

(您特意远道而来，不胜感激。)

4. 请求他人行为 (委托请求) 尊他

根据“强制性”与“委婉度”的不同，从强到弱依次为：

お/ご～ください (指示/命令)
お/ご～願います (愿望/公文)
お/ご～いただけませんか (商量/询问)
お/ご～いただければと思います (最委婉：期望/建议)

ご検討いただければと思います。

(如果您能探讨一下那就太好了。)

N2 核心考点
优先原则

日语中最常用的“去、来、吃、看、说”等高频词，拥有独立的“特殊形态”。
如果一个动词拥有“特殊敬语”形式，一般优先使用该特殊词，尽量避免生硬套用前面的通用公式（如 お行きになります）。
