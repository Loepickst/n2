💡 0. 动词的三大分类 (基础)

一切变形的前提。拿到一个动词，先看它属于哪一类。

第一类：五段动词
特征：非「る」结尾，或「る」前是 a/u/o 段。
例子：行いく、飲のむ、作つくる
第二类：一段动词
特征：以「る」结尾，且「る」前是 i/e 段。
例子：食たべる、見みる
第三类：不规则动词
特征：仅此两个。
例子：来くる (来)、する (做)
⚠️ 特殊的一类动词 (陷阱!)

特征：虽然以「iruh / eru」结尾（长得像二类），但其实是一类动词！
变形：必须按一类动词变形 (发生促音变)。

🔥 高频陷阱词表：
• 帰かえる (回) → 帰かえって
• 切きる (切) → 切きって
• 知しる (知) → 知しって
• 入はいる (进) → 入はいって
• 走はしる (跑) → 走はしって
• 要いる (需) → 要いって
• 減へる (减) → 減へって
• 滑すべる (滑) → 滑すべって
• 焦あせる (急) → 焦あせって
• 蹴ける (踢) → 蹴けって
• 喋しゃべる (聊) → 喋しゃべって
• 握にぎる (握) → 握にぎって
• 練ねる (练) → 練ねって
• 交まじる (混) → 交まじって
• 限かぎる (限) → 限かぎって
• 散ちる (散) → 散ちって
1. ます形
🔰 用法

敬体（礼貌语）。
用于对长辈、上级、陌生人说话，或者在课堂、职场等正式场合使用。表示“做...”。

🔧 规则
类型 变形规则 例子
五段 词尾 u 段 → i 段 + ます 行いく → 行いきます
一段 去掉 る + ます 食たべる → 食たべます
不规则 来る (kuru) → 来きます (kimasu)
する (suru) → します (shimasu)
💡 知识点：第一连用形 (Masu形词干)

动词变成「ます形」后，去掉「ます」剩下的部分，在语法上称为“第一连用形” (或称ます形词干)。
它是很多重要语法接续的基础：

  • 行いきます + たい → 行いきたい (想去)
  • 読よみます + ながら → 読よみながら (一边读一边...)
  • 食たべます + やすい → 食たべやすい (容易吃)
2. て形
🔰 用法

中顿 / 请求 / 进行时。
① 连接动作：去超市，然后回家。
② 请求：请做某事 (～てください)。
③ 状态：正在做某事 (～ています)。

🔧 变形规则表
类型 规则 例子
五段 (う/つ/る) 促音变 → って 待まつ → 待まって
五段 (む/ぶ/ぬ) 拨音变 → んで 飲のむ → 飲のんで
五段 (く) イ音变 → いて 書かく → 書かいて
五段 (ぐ) イ音变+浊 → いで 泳およぐ → 泳およいで
五段 (す) 无音变 → して 話はなす → 話はなして
五段 (特例) 行く (iku) → って 行いく → 行いって
一段 去掉 る + て 見みる → 見みて
不规则 来る (kuru) → 来きて (kite)
する (suru) → して (shite)
3. た形
🔰 用法

简体过去式。
表示动作已经完成。

🔧 简体规则 (た形)
⚡️ 变形秒懂技巧
变形规则与「て形」完全一致！
把 て 换成 た，で 换成 だ。
类型对照 (て → た)例子
五段 (促音) って → った 買かう → 買かった
五段 (拨音) んで → んだ 飲のむ → 飲のんだ
五段 (イ音) いて → いた 書かく → 書かいた
一段 て → た 見みる → 見みた
不规则 来きて → 来きた (读音不变)
して → した (做过)
💡 补充：敬体过去式 (ました)

上面的「た形」是简体。在正式场合必须使用敬体过去式。
规则：将「ます」换成「ました」。

  • 行いく → 行いきました (去了)
  • 来くる → 来きました (来了)
4. ない形
🔰 用法

简体否定。
1. 表示“不做...” (行かない！不去！)。

🔧 规则
类型规则例子
五段 (一般) u 段 → a 段 + ない 書かく → 書かかない
五段 (特例) 词尾为う → わ + ない 会あう → 会あわない
一段 去掉 る + ない 見みる → 見みない
不规则 来る (kuru) → 来こない (konai)
する (suru) → しない (shinai)
💡 补充：敬体否定 (ません)

上面的「ない形」是简体。在对长辈、上级说话时，要使用敬体。
否定：将「ます」换成「ません」。
过去否定：将「ます」换成「ませんでした」。

  • 行いきます → 行いきません (不去)
  • 食たべます → 食たべませんでした (没吃过/当时没吃)
🎓 知识点：未然形 (Mizenkei)

变形规则中，接续「ない」之前的部分（如 a段音）被称为“未然形”。字面意思是“尚未发生的形式”。
它是很多重要变形的基础：

  • 書かか + ない → 書かかない (否定)
  • 書かか + れる → 書かかれる (被动)
  • 書かか + せる → 書かかせる (使役)
5. 能动形
🔰 用法

能力 / 可能。
表示“具有某种能力”或“在某种条件下可行”。
⚠️ 注意：对象语的助词「を」通常要变成「が」。

🗣️ 例句：
• 日本語が話はなせます。
  (我会说日语。)
• 納豆なっとうが食たべられます。
  (我能吃纳豆。)
🔧 规则
类型规则例子
五段 u 段 → e 段 + る 行いく → 行いける
一段 去掉 る + られる 見みる → 見みられる
不规则 来る (kuru) → 来こられる (korareru)
する (suru) → できる (dekiru) *特!
💡 知识点：口语中的“去ら”现象

一段动词的标准能动形是「～られる」，但在口语中，日本人常会偷懒省掉「ら」，变成「～れる」。
例如：食たべられる → 食たべれる (能吃)。
*注意：考试或正式写作请使用标准形式。

6. 意志形
🔰 用法

意向 / 提议。
1. 独白：表达决心，打算做某事 (......ようと思う)。
2. 劝诱：邀请对方“一起做吧！”。

🔧 规则
类型规则例子
五段 u 段 → o 段 + う 行いく → 行いこう
一段 去掉 る + よう 始はじめる → 始はじめよう
不规则 来る (kuru) → 来こよう (koyou)
する (suru) → しよう (shiyou)
💡 补充：敬体形式 (ましょう)

上面的「意志形」是简体。在对长辈或正式场合表达提议，请使用「ましょう」。

  • 行いこう → 行いきましょう (走吧)
7. 被动形
🔰 用法

主语承受动作。
主要分为以下三种情况：

  • 1. 直接被动：被别人做了某事。
    母に褒ほめられました。
    (被妈妈表扬了。)
  • 2. 受害被动：遭受了麻烦 (含“受害”语感)。
    雨に降ふられました。
    (被雨淋了 / 遭雨淋。)
  • 3. 公共被动：描述客观事实 (不强调施动者)。
    この本ほんは読よまれています。
    (这本书被(世人)阅读。)
🔧 规则
类型规则例子
五段 u 段 → a 段 + れる 踏ふむ → 踏ふまれる
一段 去掉 る + られる 褒ほめる → 褒ほめられる
不规则 来る (kuru) → 来こられる (korareru)
する (suru) → される (sareru)
💡 小贴士：撞车了！

一段动词的被动形和能动形长得一模一样！
例如：「食たべられる」既可以是“能吃”，也可以是“被吃”。

8. 使役形
🔰 用法

强制 / 许可。
表示“让某人做...”或者“允许某人做...”。

  • 1. 强制 ：上级命令下级。
    部下ぶかを働はたらかせる。
    (让部下去工作。)
  • 2. 许可 ：允许对方做某事。
    子供こどもにアイスを食たべさせる。
    (让/准许孩子吃冰淇淋。)
🔧 规则
类型规则例子
五段 u 段 → a 段 + せる 待まつ → 待またせる
一段 去掉 る + させる 食たべる → 食たべさせる
不规则 来る (kuru) → 来こさせる (kosaseru)
する (suru) → させる (saseru)
9. 使役被动形
🔰 用法

被迫... / 不得不...
直译：“被（别人）强行让做了...”。
表达一种“不情愿、被迫”的心情。

🗣️ 例句：
• 母に野菜を食たべさせられました。
  (被妈妈逼着吃了蔬菜。)
• 待またされました。
  (被迫等待 / 让人好等。)
🔧 规则
⚡️ 组合技： 使役形 (させる) + 被动 (られる)
五段动词通常使用缩略形 (～される)。
类型规则例子
五段 (缩略形) u 段 → a 段 + される 行いく → 行いかされる
一段 去掉 る + させられる 食たべる → 食たべさせられる
不规则 来る (kuru) → 来こさせられる
する (suru) → させられる
⚠️ 避坑：以「す」结尾的动词

五段动词中，如果词尾是「す」(如：話す)，不能用缩略形！必须用完整形式。
❌ 話される (错！这是被动形)
⭕ 話はなさせられる (被迫说)

10. 命令形
🔰 用法

强行命令。
表示“快做！”“给我做！”。
⚠️ 语气非常粗鲁，仅用于紧急情况、军队、或者男性吵架等。请勿对长辈使用。

🔧 规则
类型规则例子
五段 u 段 → e 段 行いく → 行いけ (滚/去!)
一段 去掉 る + ろ 見みる → 見みろ (看!)
不规则 来る (kuru) → 来こい (koi)
する (suru) → しろ (shiro)
🛑 补充：禁止形 (不准做!)

与命令形对应的是“禁止形”。
规则：辞书形 (原型) + な

  • 行いくな！ (别去！)
  • 食たべるな！ (别吃！)
  • 負まけるな！ (别输！/ 加油)
11. 假定形 (ば形)
🔰 用法

如果...的话 (条件)。
表示假设条件。常用于“如果发生A，就会发生B”。
也可以用于提建议（...ばいい：...做就好）。

🔧 规则
类型规则例子
五段 u 段 → e 段 + ば 行いく → 行いけば
一段 去掉 る + れば 見みる → 見みれば
不规则 来る (kuru) → 来くれば (kureba)
する (suru) → すれば (sureba)
形容词 (特殊) 词尾 い → ければ 高たかい → 高たかければ
否定形 (特殊) 词尾 ない → なければ 行いかない → 行いかなければ
💡 常用句型：不得不...

将否定形变成假定形，再加上「ならない」，就变成了必须做某事。
公式：...なければ + なりません
勉強べんきょうしなければなりません。(必须学习 / 不学不行)
